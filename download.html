Install

Get started in under a minute

SenseAI runs on Chrome, Edge, Brave, and any Chromium-based browser. Follow these steps to install.

Download ZIP from GitHub

Download the extension

Click the download button above. You'll get a .zip file containing all extension files.

Unzip the folder

Extract the ZIP file to a permanent location on your computer (e.g., your Documents folder). Don't delete this folder ‚Äî Chrome needs it.

Open the extensions page

Open Chrome and go to chrome://extensions in the address bar. Or go to Menu (three dots) → Extensions → Manage Extensions.

Enable Developer Mode

In the top-right corner of the extensions page, toggle on Developer mode.

Load the extension

Click "Load unpacked" in the top-left corner, then select the unzipped folder. SenseAI will appear in your extensions list.

Allow microphone access

Visit any website and allow microphone access when prompted. Say "Hey Sense" ‚Äî you're ready to go.

FAQ

Frequently asked questions

Quick answers to common questions about SenseAI.

Which browsers are supported?

SenseAI works on Chrome, Microsoft Edge, Brave, Opera, and any Chromium-based browser that supports the Web Speech API. Firefox and Safari are not yet supported due to differences in speech recognition APIs.

Is it free?

Yes, completely free and open source. No accounts, no subscriptions, no data collection.

Does it send my voice data anywhere?

SenseAI uses your browser's built-in speech recognition (Web Speech API). Voice processing is handled by your browser ‚Äî the extension itself does not transmit or store any audio or transcripts.

Can I use it without saying the wake word?

Yes. Press Ctrl+Shift+S (or Cmd+Shift+S on Mac) to activate the assistant instantly with your keyboard.

Why does it ask for microphone permission on every site?

Chrome requires microphone permission per-site for security. You can allow it permanently for trusted sites in Chrome's site settings.

The assistant isn't hearing me. What should I do?

Make sure microphone access is allowed for the current site (check the lock icon in the address bar). Try refreshing the page. If using a Bluetooth headset, make sure it's set as the default input device in your system settings.

Can I use it on mobile?

Not currently. Chrome extensions are only supported on desktop browsers. Mobile browser support is a future goal.
